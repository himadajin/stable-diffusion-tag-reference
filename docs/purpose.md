# Stable Diffusion Tag Reference

## Purpose

ComfyUI で Stable Diffusion のプロンプトを編集するときに、どのようなタグがあるのかを調べ、目的の英語タグへ辿り着いてコピーしやすくする参照サイトを作る。

このリポジトリの目的はタグの追加、編集、再分類、整理作業ではない。元データに含まれるカテゴリ構造とタグ情報を活かし、制作中にタグを探すための見通しを良くすることを目的とする。

## Context

`tmp/prompt-alchemy-assets` に Prompt Alchemy 由来の仮置きファイルがある。主なデータは、カテゴリ付きの `tags.json` と、自由入力候補の `free-input-tags.json`。

`tags.json` には日本語の大カテゴリとサブカテゴリ階層があり、タグには主に英語タグと日本語名が含まれる。大カテゴリは用途ごとに十分分かれており、「身体かつ食べ物」のような横断分類を前提にしなくても、カテゴリ単位でページを分けた方が使いやすい見込みがある。

サイトの利用者は、ComfyUI でプロンプトを調整している最中に、日本語のカテゴリや説明を手がかりにタグを探し、ComfyUI に貼るための英語タグをコピーしたい。

データ形式は後で扱いやすく変換してもよい。ただし、その変換は参照性を上げるためのものであり、このリポジトリ内でタグデータ自体を編集・管理することを目的にしない。

## Direction

既存の大カテゴリをサイドバーに並べ、カテゴリごとに別ページまたは明確に分かれたビューとして表示する。各カテゴリ内では、元データのサブカテゴリ階層を基本的に維持し、利用者が階層を眺めながら目的のタグへ辿れるようにする。

検索は主導線ではなく補助導線として扱う。日本語名、英語タグ、自由入力候補を検索できるようにし、検索結果にはヒットしたタグだけでなく、大カテゴリやサブカテゴリなどの文脈も一緒に示す。検索結果から同じ分類内の近いタグを探し直せることを重視する。

コピー対象は英語タグを主とする。日本語名やカテゴリ名は理解と探索の補助として扱い、日本語から探して英語タグを迷わずコピーできる状態を目指す。

後続の設計や実装では、タグ編集ツールやプロンプト作成ツールへ広げすぎない。まずは、元データを活かした読みやすいタグ参照サイトとして成立させる。
